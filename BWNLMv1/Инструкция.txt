1. Установить SQL server
2. Установить python не ниже 3.4 версии
3. Поместить папку BWNLMv1 в корень диска С
4. Открыть панель управления, зайти в разел администрирование и выбрать Источники данных ODBC
5. Добавить пользовательский DSN - выбрать SQL server - в имени DSN указать BWNLM - сервер SQL указать тот с которым осущеествляется соединение программы BONWINlock-
-в способе аутентификации выбрать проверку подлинности SQL - снять галку с "Получить параметры, используемые по умолчанию, от SQL server"
6. Остановить работу BWNLMServer
7. Закрыть программу BonWinLock system
8. Запустить run.bat


протестированно с SQL server 2012, 2017
python 3.4 и 3.8.1
!!!!!!Программа настроена на базу данных BWNLM2_db если у вас бд называется подругому, вам нужно редактировать файл sql_connect.py!!!!!!
!!!!!!Для удаленного подключения редактируй sql_connect.py!!!!!!
